# DOT Compliance Dashboard

## Features
- Real-time tracking of DOT compliance metrics
- User-friendly dashboard interface for easy navigation
- Integration with third-party compliance tools
- Customizable reports and alerts
- Historical data analysis for trend identification

## Setup Instructions
1. **Clone the Repository**  
   ```bash
   git clone https://github.com/chellgeth/Lyle.git
   cd Lyle
   ```
2. **Install Dependencies**  
   Make sure you have all required dependencies installed:
   ```bash
   npm install
   ```
3. **Configuration**  
   Create a `.env` file in the root directory and add the necessary environment variables.
4. **Run the Application**  
   Start the application with:
   ```bash
   npm start
   ```
5. **Access the Dashboard**  
   Open your browser and go to `http://localhost:3000` to access the DOT compliance dashboard.
